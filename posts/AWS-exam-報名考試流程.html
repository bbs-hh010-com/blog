认证指南

AWS 认证考试报名全流程

从注册账号到预约考试，一步步带你拿下 AWS 认证，清晰、全面、不踩坑。

1

考前准备

在正式开始报名之前，请确保你已经准备好以下材料——这会让整个流程顺畅很多。

  • 一个有效的电子邮箱（用于注册 AWS Certification 账号）
  • 一张有效的国际信用卡（Visa / MasterCard / 银联）或考试券
  • 政府颁发的带照片的身份证件（护照、身份证、驾照等）
  • 稳定的网络环境（如果选择在线考试）
  • 静音、无干扰的考试空间（在线考试需满足环境要求）
💡 温馨提示： 建议提前准备好身份证件的电子扫描件或清晰照片，部分考试中心可能会要求上传。
2

注册 AWS Certification 账号

A
访问 AWS Certification 官网
打开浏览器，访问 aws.amazon.com/certification，点击页面右上角的 "Sign In to the AWS Certification Account"。
B
创建或登录 AWS 账号
如果你已有 AWS 根账号（用于使用 AWS 云服务的账号），可以直接使用该邮箱登录。如果没有，点击 "Create a new AWS account" 创建一个新账号。注意：AWS 根账号与 AWS Certification 账号绑定，但认证账号有独立的档案系统。
C
完善个人信息
登录 Certification 平台后，进入 "My Account" 页面，填写或确认你的姓名、邮箱、联系电话和邮寄地址。姓名必须与考试时出示的身份证件完全一致，否则可能导致无法入场。
⚠️ 重要提醒： 姓名拼写必须和护照/身份证上的 拼音/英文名 完全一致。如果有中间名，务正确实填写。不一致会导致考试中心拒绝入场。
3

选择考试

AWS 提供多种认证路径，涵盖不同角色和专业领域。以下是主流认证体系：

认证级别 代表考试 建议备考时间 考试费用
☁️ Foundational Cloud Practitioner (CLF-C02) 2-4 周 $100 USD
⚙️ Associate Solutions Architect (SAA-C03) 6-12 周 $150 USD
⚙️ Associate Developer (DVA-C02) 6-10 周 $150 USD
⚙️ Associate SysOps Administrator (SOA-C02) 6-10 周 $150 USD
🚀 Professional Solutions Architect Professional (SAP-C02) 10-16 周 $300 USD
🔬 Specialty Security / Data Analytics / ML 等 8-12 周 $300 USD
🌟 新手推荐： 如果你是 AWS 新手，建议先从 AWS Certified Cloud Practitioner (CLF-C02) 入手，覆盖面广、难度适中，适合建立整体认知。
4

预约考试

1
登录 Certification 平台，点击 "Schedule an Exam"
在 AWS Certification 后台，点击 "Schedule New Exam" 按钮，选择你要参加的考试名称和版本（如 AWS Certified Solutions Architect – Associate SAA-C03）。
2
选择考试模式
你可以在以下两种模式中选择：
🏢 线下考试中心 (Pearson VUE Test Center) — 前往授权考试中心，在监考人员监督下完成考试。
💻 在线监考考试 (Online Proctored Exam) — 在家或办公室通过网络摄像头进行，由远程监考人员实时监督。
3
选择时间和地点
考试中心模式：输入你的城市或邮编，系统会列出附近的考试中心和可选时间。
在线模式：选择你偏好的日期和时间段（通常 24/7 都有可用考位）。
4
确认并支付
核对考试信息、时间、地点无误后，进入支付环节。支持 信用卡（Visa / MasterCard / 银联） 支付，也可以输入 考试券（Exam Voucher） 兑换码。支付完成后，你会收到一封确认邮件。
📧 确认邮件： 预约成功后，AWS 和 Pearson VUE 都会发送确认邮件到你的注册邮箱。请务必保存好邮件中的 考试确认编号（Confirmation ID）。
5

考前注意事项

✓
证件准备
必须携带 政府颁发的、带有照片的身份证件原件（护照、身份证、驾照均可）。证件上的姓名必须与 Certification 账号中的姓名 完全匹配。
✓
提前到达
线下考试中心：建议提前 30 分钟 到达考点，完成签到和身份验证流程。
在线考试：建议提前 15 分钟 登录系统，运行系统检查并完成环境扫描。
✓
在线考试环境要求
房间要求： 独立、安静、无干扰。桌面必须清空，不能有书籍、纸张、手机等物品。
设备要求： 电脑需配备摄像头和麦克风，安装 Pearson VUE 安全浏览器。
着装要求： 穿着得体，监考人员需要全程看到你的面部和双手。
✓
改期与取消
考试前 24 小时 以上可以免费改期或取消。不足 24 小时则无法更改，考试费用不予退还。
修改方式：登录 Certification 平台 → "My Account" → "Manage Scheduled Exams" 进行操作。
⚠️ 在线考试特别注意： 严禁佩戴耳机、帽子、墨镜；考试过程中不能有人进入房间；不能查看手机或其他电子设备；不能随意离开摄像头视野。违反规则可能导致考试被终止。
6

考试当天流程

🕐
签到 & 身份验证
线下：到前台出示身份证件，签署保密协议（NDA），存放个人物品，进入考试间。
线上：启动 Pearson VUE 安全浏览器，拍摄身份证件，扫描考试环境（摄像头旋转 360° 展示房间）。
📋
阅读 NDA 与教程
考试开始前，你需要阅读并同意 保密协议（NDA），然后进入一个可选的考试教程界面，熟悉考试系统的操作。
⏱️
正式考试
考试为 计时模式，屏幕上会显示剩余时间。你可以标记不确定的题目并回头检查。大多数 AWS 考试包含 65 道题（选择题 + 多项选择 + 拖拽题等），时长 130 分钟。部分考试有额外时间（如 CLF-C02 为 90 分钟 / 50 题）。
🎯
查看成绩
提交答卷后，立即 在屏幕上看到 通过/未通过 的结果以及分数区间。详细的各领域成绩报告将在 5 个工作日内 发送到你的注册邮箱。
🎉 通过之后： 你可以在 Certification 平台下载电子版证书和数字徽章（通过 Credly 发放）。证书有效期为 3 年，之后需要重新认证（renew）。
7

常见问题 FAQ

?
没有 AWS 根账号可以考认证吗？
不可以。AWS Certification 账号需要关联一个 AWS 根账号才能使用。注册 AWS 根账号是免费的，不需要购买任何服务。
?
考试券在哪里购买？
可以通过 AWS 官网购买，也可以在 AWS 合作伙伴处购买，或参加 AWS 官方活动获取折扣券。部分第三方培训机构也提供考试券团购服务。
?
考试可以使用中文吗？
部分考试 提供简体中文界面和/或中文语音题目标注（auditory instructions）。例如 Cloud Practitioner (CLF-C02) 和 Solutions Architect Associate (SAA-C03) 均支持中文。建议预约时确认语言选项。
?
没通过怎么办？可以重考吗？
可以重考。首次未通过后需等待 14 天 才能再次参加同一考试。之后每次重考同样间隔 14 天。需要重新支付考试费用。
?
考试结果多久正式出？
屏幕上立即显示通过/未通过结果，正式的详细成绩报告在 5 个工作日内 通过邮件发送。证书和数字徽章在成绩公布后即可下载。
